
# Printscript

### Pre-commit Script
To copy pre-commit script to .git/hooks run:
```bash
./gradlew installGitHook
```

### Linter check
To check linter:
```bash
./gradlew check
```

## Run Code
You can run PrintScript code using:
```bash
./print-script execute <source>
```
For help run:
```bash
./print-script --help
```
